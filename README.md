# Backend — Pedido Fácil

## Instalação
```bash
npm install
```

## Configuração
Copie o `.env.example` para `.env` e preencha as variáveis:
```bash
cp .env.example .env
```

## Rodar
```bash
npm run dev   # desenvolvimento
npm start     # produção
```

## Variáveis de ambiente
- `DATABASE_URL` — string de conexão PostgreSQL
- `PORT` — porta do servidor (padrão: 3001)
